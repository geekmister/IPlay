# IPlay 内容系统起步

你现在看到的页面来自一个 `.mdx` 文件。

这让工具站可以继续保持轻量，同时获得内容能力：

- 用 Markdown 写教程和案例
- 用 Frontmatter 管理标题、摘要、发布时间
- 后续可以逐步加入交互组件

## 当前最小流程

1. 在 `content-src/posts` 下新增 `.mdx`
2. 运行 `npm run build:mdx`
3. 打开 `content/index.html` 查看文章列表

## 适合先写哪些内容

- 工具使用教程
- 常见问题与踩坑
- 场景案例（电商图、人像图、证件照等）

## 下一步可以扩展

- 标签与分类页
- RSS 与 sitemap
- 自动内链与相关文章推荐
