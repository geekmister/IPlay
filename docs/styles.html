05 main.css 样式说明

下面列出 `assets/css/main.css` 中最重要的样式类，并给出真实使用示例与说明，保证每个核心类都有配套参考。

.btn
基础按钮类，提供统一圆角、内边距和 hover 动画。用于所有页面内的操作按钮。
.btn-primary
主操作按钮，如“执行”、“开始生成”、“下载结果”。
.btn-secondary
次级按钮，适用于“重置”、“删除”、“导出”之类辅助操作。
.btn-danger
危险动作按钮，用于“删除”、“清空”等高风险操作。
.btn-vip
VIP 风格按钮，适用于“高级玩法”页面的重点入口。
.upload-box
上传区域样式，带虚线边框、圆角和中心提示。用于图片上传模块。
.upload-box--filled
已上传后状态，去掉 hover 强调，表示已填充内容。
.image-preview
图片预览容器，内部图片居中展示。用于上传后预览区。
.workspace
上传后工作区外层容器，提供垂直间距，连接各模块。
.feature-panel
两列功能区域布局，左侧为功能列表，右侧为功能详情。
.feature-btn
功能选择按钮，默认灰边，激活态主色背景。用于人像/增强/隐私页面。
.feature-detail
功能详情面板，显示当前选中功能的说明与执行按钮。
.result-panel
结果展示区域，浅灰背景、圆角，显示处理后图片或文本结果。
.action-bar
底部操作栏，放置“恢复原图”、“删除”、“下载”等按钮。
.form-control
输入框、选择框、文本域统一样式，用于所有表单字段。
.form-label
表单标签样式，用于标题和字段说明。
.badge
小标签，用于标记“可选”、“VIP”等状态。
.grid-2 / .grid-3 / .grid-4
响应式网格布局，用于功能卡片、示例面板、列表排列。
.workflow-grid
批量与输出页面四卡片布局。
.workflow-card
工作项卡片，含图标、标题、说明和操作表单。
.category-grid
高级玩法分类按钮的网格布局。
.category-btn
分类选择按钮，默认白底，激活后主色边框与浅色背景。
.card
通用卡片样式，用于信息面板、说明区域、配置卡片。
.page-title
页面主标题，使用 1.5rem / 700，颜色 #111827。
.page-subtitle
页面副标题，使用 0.875rem / 400，颜色 #6b7280。

06 详细规范

间距体系

核心间距：8 / 16 / 24 / 32 / 40。页面内部间距优先采用 24px 和 32px。

颜色与层次

主色用于重点按钮、激活态和交互强调；灰色用于文本、分割线和背景层次。

交互状态

按钮 hover 使用轻微透明度变化；禁用态为 50% 透明度。
